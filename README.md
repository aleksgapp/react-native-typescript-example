## Node and dependency versions
At the time of this readme
- Node 8.2.1
- npm 5.3.0
- Typescript 2.4.2

## Dependencies
```bash
brew install node
brew install watchman
sudo npm install -g react-native-cli semistandard semistandard-format yarn
```

## Git instructions

```
cd <project>
npm install
```

## Running the project
Terminal 1 (Compiles typescript)
```
npm run tsc
```

Terminal 2
```
npm start
```

Terminal 3
```
npm test
npm run ios
npm run android
```
